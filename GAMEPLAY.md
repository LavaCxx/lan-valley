# GAMEPLAY.md - LanValley (TUI Edition)

## 1. 游戏愿景 (Concept)
一款运行在终端（Terminal）里的“像素风”种田游戏。
它是一个**静默运行的桌面盆景**。利用公司内网 IP 的最后一位数字，将每个人的电脑模拟成不同气候的“农场”。
玩家在工作间隙（<1分钟操作）照料作物，并与局域网内的同事交换特产，制作料理。

---

## 2. 核心机制 (Core Mechanics)

### 2.1 IP 气候带 (The IP Climate)
根据本机 IP 地址的哈希值（或末位），生成不可更改的“原生气候”。

*   **Modulo 0: 春之平原 (Spring Plains)**
    *   **特产作物**：防风草(Parsnip)、土豆(Potato)、绿豆(Green Bean)。
    *   **视觉风格**：淡绿色调，温和。
*   **Modulo 1: 夏之海岛 (Summer Isles)**
    *   **特产作物**：蓝莓(Blueberry)、甜瓜(Melon)、辣椒(Hot Pepper)。
    *   **视觉风格**：明亮，水系丰富。
*   **Modulo 2: 秋之矿山 (Autumn Hills)**
    *   **特产作物**：南瓜(Pumpkin)、蔓越莓(Cranberry)、红薯(Yam)。
    *   **视觉风格**：深褐色，枯黄。

**痛点设计（强制联机）**：
*   **目标**：制作【全家桶料理】(Complete Breakfast) 给机器人升级。
*   **配方**：需要【土豆煎饼】(来自平原) + 【甜瓜汁】(来自海岛)。
*   **玩法**：你只能种出土豆，必须挂机派“货车”去拥有海岛气候 IP 的同事那里交换甜瓜。

### 2.2 种植循环 (Farming Loop)
*   **基础操作**：耕地 -> 播种 -> 浇水 -> 收获 -> (放入出货箱换金币 / 放入加工桶做果酱)。
*   **自动化 (Automation)**：
    *   **T1 洒水器**：每天早上自动浇灌周围 4 格。
    *   **T2 祝尼魔小屋 (Junimo Hut)**：类似 Rustys 的机器人，自动收割成熟作物并入库。
    *   **挂机收益**：程序后台运行时，作物依然生长。早上上班打开，正好收成。

### 2.3 局域网集市 (LAN Market)
*   **发现邻居**：TUI 右侧面板自动列出局域网内开启游戏的同事：
    *   `[ONLINE] 192.168.1.55 (夏之海岛)`
    *   `[ONLINE] 192.168.1.12 (秋之矿山)`
*   **货车系统**：
    *   选择一个邻居，装填 50 个【防风草】，点击“发送”。
    *   货车出发（倒计时 10 分钟）。
    *   到达后，对方库存自动增加防风草，你的货车自动带回对方设置的“回礼”（或者直接卖给对方换金币）。
